# Sales & Profit Dashboard

## Tools Used
- Power BI
- Excel

## Project Overview
This dashboard analyzes sales, profit, customer segments, payment modes, and regional performance.

## Key Insights
- Consumer segment generated the highest sales.
- COD was the most used payment mode.
- Q4 showed the strongest profit growth.
- Technology and Office Supplies performed strongly.
